# Serenity Product Asset Workflow

## Source-of-truth assets
- Keep editable/source product renders in `images/products/*.png`.
- Keep bundle/stack renders in `images/stacks/*.png`.
- Generated WebP files live beside the PNGs and are safe to regenerate.

## Optimization
Run from the project root:

```bash
python3 scripts/optimize_assets.py
```

The script uses `cwebp` when available, falling back to ImageMagick `magick`.
It creates:

- `images/products/*.webp`
- `images/stacks/*.webp`

## Label workflow
- Print-ready vial labels are generated by `labels/generate_labels.py`.
- Preview labels at `labels/preview.html`.
- Update product name/weight/form/purity in `labels/generate_labels.py`, then rerun:

```bash
python3 labels/generate_labels.py
```

## Supplier handoff placeholders
Before production, replace placeholder/source renders with supplier-confirmed:

- Final vial dimensions and label wrap size
- Final lot-number format
- Final COA/lab-report PDF per SKU under `coa/`
- Final storage and handling language from supplier
- Final product photography or 3D renders
